## gitbook 安装

#### 1、搭建nodejs环境
	官网下载 https://nodejs.org/en/

	查看是否安装成功
	node -v 
	v10.23.0

#### 2、安装gitbook
	更好node镜像源
	npm install -g cnpm --registry=https://registry.npm.taobao.org

	安装gitbook-cli
	npm install -g gitbook-cli

	查看当前已经安装的gitbook版本
	gitbook versions

	启动gitbook
	gitbook serve

	浏览器访问
	http://127.0.0.1:4000
